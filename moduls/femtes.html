Diari intestinal

Seguiment de deposicions, tipus Bristol i evolució intestinal.

Què permet registrar

El mòdul de diari intestinal permet registrar de forma senzilla les deposicions, el tipus segons l’ i les observacions relacionades amb el ritme intestinal.

Què aporta al seguiment

El registre continu ajuda a identificar canvis en la freqüència, consistència i regularitat intestinal, facilitant una visió més objectiva de l’evolució.

Aquesta informació pot ser útil per comentar hàbits intestinals, canvis de dieta, medicació o símptomes amb professionals sanitaris.

Base clínica

El mòdul pren com a referència l’escala de Bristol, una eina utilitzada per classificar les femtes en set tipus segons la seva forma i consistència. Aquesta escala facilita descriure el patró intestinal amb un llenguatge comú entre pacient i professional sanitari.

Diverses guies i materials clínics utilitzen l’escala de Bristol per valorar la consistència de les deposicions i fer seguiment de problemes com restrenyiment o diarrea.

Informes i gràfics

Les dades registrades es poden consultar amb gràfics i informes resum, facilitant la revisió de l’evolució intestinal durant el període seleccionat.
